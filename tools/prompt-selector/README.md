# Prompt Selector Tool

## Purpose

Tool for selecting appropriate prompts based on task and context.

**Prompt selector enables optimal prompt selection.**

---

## Selection Process

### Step 1: Analyze Task

**What to Analyze**:
- Task type
- Task complexity
- Task context
- Task requirements

**Output**: Task analysis

---

### Step 2: Identify Mode

**What to Identify**:
- Required mode
- Mode sequence
- Mode dependencies
- Mode transitions

**Output**: Mode identification

---

### Step 3: Select Prompts

**What to Select**:
- Master prompt
- Mode prompts
- Task prompts
- Chain prompts

**Output**: Prompt selection

---

## Selection Criteria

### Task-Based Selection

**Criteria**:
- Task type matches prompt
- Task complexity matches prompt
- Task context matches prompt

**Why**: Task-based selection ensures relevance.

---

### Mode-Based Selection

**Criteria**:
- Mode matches workflow
- Mode sequence correct
- Mode dependencies met

**Why**: Mode-based selection ensures workflow.

---

## Usage

### When to Select

**Select when**:
- Starting new task
- Changing modes
- Complex workflows
- Uncertain prompt choice

**Why**: Selection ensures optimal prompts.

---

### How to Use

**Usage process**:
1. Describe task
2. Run selector
3. Review prompt suggestions
4. Use selected prompts
5. Chain if needed

**Why**: Systematic usage ensures quality.
