# Loop Analyzer Tool

## Purpose

Tool for analyzing core loop effectiveness, identifying issues, and suggesting improvements.

**Loop analysis ensures loops are effective and engaging.**

---

## Analysis Dimensions

### Loop Clarity

**What to Analyze**:
- Is loop clear?
- Are steps obvious?
- Is flow smooth?
- Is purpose clear?

**Why**: Clarity enables effective loops.

---

### Loop Engagement

**What to Analyze**:
- Does loop create engagement?
- Is loop satisfying?
- Does loop maintain interest?
- Is loop rewarding?

**Why**: Engagement ensures loop effectiveness.

---

### Loop Progression

**What to Analyze**:
- Does loop create progress?
- Is progress meaningful?
- Does progress feel good?
- Is progress visible?

**Why**: Progression maintains motivation.

---

## Analysis Output

### Analysis Report

**Report Includes**:
- Loop clarity assessment
- Engagement analysis
- Progression evaluation
- Issue identification
- Improvement recommendations

**Why**: Comprehensive analysis enables improvement.

---

## Integration Points

### Core Loop Evolution
- Analyzes loop effectiveness
- Identifies evolution needs
- Suggests evolution paths

### Design Intelligence
- Validates against design layers
- Ensures loop alignment
- Supports loop design

---

**This tool enables systematic loop analysis and improvement.**
