# Game Analysis Template

## Purpose

Template for analyzing open-source games, extracting mechanics, systems, and design patterns.

**This template ensures systematic game analysis.**

---

## Analysis Structure

### Basic Information

```markdown

**Repository**: [URL]
**License**: [License]
**Language**: [Language]
**Status**: [Status]

**Genre**: [Genre]
**Platform**: [Platform]
**Complexity**: [Simple/Medium/Complex]
```

Analysis is complete when:
- ✅ All sections filled
