# Psychology Audit Checklist

## Purpose

Checklist for auditing psychology implementation, ensuring designs respect cognitive limits and serve motivation.

**Psychology audits ensure psychological alignment.**

---

## Audit Checklist

### Motivation Loops
- [ ] Loops serve user motivation
- [ ] Loops are clear and effective
- [ ] Rewards are timed appropriately
- [ ] Progress is visible and meaningful

### Cognitive Load
- [ ] Cognitive load is manageable
- [ ] Information is organized clearly
- [ ] Choices are limited appropriately
- [ ] Learning is supported

### Reward Timing
- [ ] Rewards are immediate for feedback
- [ ] Rewards are timed appropriately
- [ ] Reward variety exists
- [ ] Rewards maintain value

### Failure States
- [ ] Failures are handled gracefully
- [ ] Recovery is possible
- [ ] Failure feedback is clear
- [ ] Failures don't frustrate excessively

---

**This checklist ensures psychological alignment through systematic auditing.**
