# Glitched Fusion Mechanics & Rule-Breaking Abilities

## Overview
This document details the mechanics and abilities associated with glitched fusion in Pixel Pets Reborn X Remerged. Glitched fusion occurs when Chaos stones (Opal stones) are used in the fusion process, creating pets with rule-breaking abilities that bypass normal game mechanics.

## Glitched Fusion Trigger Conditions
- **Standard Chance**: 3-7% chance when Chaos stones are used in fusion
- **Enhanced Chance**: Up to 15% chance when two Chaos stones are used (Opal + Opal fusion)
- **Maximum Chance**: 25% chance when using glitched Chaos stones (see "Glitched Stone Variants" section)
- **Fusion Intent Synergy**: CORRUPTION intent increases glitch chance by 5%

## Core Glitched Fusion Mechanics

### 1. Rule-Breaking Properties
Glitched pets possess abilities that bypass normal game mechanics:
- **Ignore Defense**: Abilities can bypass up to 100% of target's defense
- **Bypass Resistance**: Resistances to status effects can be ignored
- **Infinite Loops**: Create temporary conditions that loop infinitely (with safety limits)
- **Logic Reversal**: Reverse the effects of enemy abilities
- **System Override**: Temporarily override game rules for specific scenarios

### 2. Unpredictable Effects
- **Variable Outcomes**: Same ability can have different effects each use
- **Chaos Dice**: Random chance tables that determine ability behavior
- **Reality Fluctuation**: Stats and abilities may change during battle
- **Probability Distortion**: Manipulate chance-based mechanics

### 3. Glitch Manifestation Levels
- **Minor Glitch**: Subtle rule bending (e.g., extra 10% damage)
- **Major Glitch**: Significant rule breaking (e.g., ignore half of defense)
- **Critical Glitch**: Complete rule override (e.g., bypass all defense)
- **Reality Break**: Entire game rules temporarily suspended

## Glitched Pet Classes

### Class 1: Bypass Specialists
Pets that specialize in bypassing specific game mechanics:
- **Defense Bypassers**: Ignore physical/magical defense
- **Status Immunity Breakers**: Apply status effects to immune targets
- **Cooldown Ignorers**: Use abilities without cooldowns
- **Energy Manipulators**: Use abilities without energy cost

### Class 2: Reality Distorters
Pets that alter fundamental game mechanics:
- **Time Dilation**: Change turn order or battle speed
- **Probability Shifters**: Manipulate chance-based events
- **Stat Modifiers**: Alter fundamental stats permanently
- **Logic Inverters**: Reverse all effects (damage becomes healing, etc.)

### Class 3: Chaos Engines
Pets that create unpredictable effects:
- **Random Generators**: Cause random effects each turn
- **State Shufflers**: Randomly swap stats, abilities, or positions
- **Effect Mutators**: Change the nature of abilities in battle
- **Outcome Reversers**: Reverse battle outcomes

### Class 4: System Overrides
Pets that temporarily change game rules:
- **Rule Rewriters**: Change battle conditions for limited time
- **Domain Glitchers**: Create areas where normal rules don't apply
- **Reality Anchors**: Force specific conditions to remain active
- **Paradox Creators**: Create logical impossibilities

## Glitched Fusion Process

### Pre-Fusion Glitch Detection
1. Check if Chaos stones are being used
2. Calculate glitch chance based on stone tiers and types
3. Apply fusion intent modifiers
4. Roll for glitch occurrence (3-7% base)

### Glitch Application During Fusion
1. If glitch occurs, mark resulting pet as "Glitched"
2. Increase mutation count by 1 (affects sprite generation)
3. Apply rule-bending properties based on parent elements
4. Generate glitched abilities instead of standard ones
5. Add chaos-specific lore elements
6. Mark in lineage as glitched for future fusion

### Post-Fusion Glitch Validation
1. Ensure glitched pet doesn't break game entirely
2. Set reasonable limits on rule-breaking abilities
3. Maintain balance against non-glitched pets
4. Log fusion for uniqueness scoring (glitched pet bonus)

## Glitched Pet Limitations & Balance

### Built-in Safeguards
- **Glitch Stability**: Higher mutation count = less stable abilities
- **Rule Limiters**: Cannot break core game functions (win/loss conditions)
- **Power Checks**: Glitched pets have energy/turn limits
- **Counter Mechanisms**: Non-glitched pets have anti-glitch abilities

## Implementation Notes

### Technical Considerations
- Glitched abilities should have unique visual effects (glitch artifacts, color shifting)
- All bypass mechanics need validation to prevent game-breaking
- Random effects should be seeded to ensure reproducibility
- Anti-glitch measures must be balanced to prevent nullification

### Game Balance
- Glitched pets should be rare and difficult to obtain
- Counter-abilities should exist in the game for balance
- Glitch effects should have reasonable limitations
- Higher-tier pets may be needed to effectively use anti-glitch

### Player Experience
- Glitched pets should be clearly identified in collection
- Fusion chance should be displayed when attempting fusion
- Uniqueness scores should account for glitched properties
- Visual indicators should show glitch effects in battle
