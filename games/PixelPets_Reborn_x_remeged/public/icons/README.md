# PWA Icons

This directory should contain the following icon files for the PWA manifest:

- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels

## Generating Icons

You can generate these icons using the HTML generator:

1. Open `icon-generator.html` in your browser
2. Click the "Generate Both" button
3. The icons will be downloaded automatically
4. Move them to this directory

Alternatively, create the icons manually using any image editor:
- Theme color: `#1a1a2e` (dark blue)
- Accent color: `#00ffff` (cyan)
- Recommended: Simple logo or "PP" text on colored background
