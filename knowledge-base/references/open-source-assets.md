# Open Source Asset Sources

## Purpose

This document catalogs **free, open-source asset sources** - where to find graphics, audio, and other assets for projects. All sources listed are free to use (check individual licenses).

---

## Graphics & Sprites

### Kenney.nl

**URL**: https://kenney.nl/assets

**Description**: High-quality game assets including sprites, UI elements, and 3D models. Most assets are CC0 (public domain).

**License**: CC0 (public domain) for most assets

**Best For**: 
- Game sprites
- UI elements
- Icons
- Complete asset packs

**Notes**: 
- Very high quality
- Consistent style
- Regular updates
- Multiple themes available

## How to Use This Library

### When Sourcing Assets

1. **Check license** - Ensure compatible with project
2. **Verify quality** - Check asset quality
3. **Download and organize** - Store in `/src/assets`
4. **Document source** - Keep track of sources
5. **Attribute if required** - Follow license requirements

### When Adding New Sources

1. **Verify free/open-source** - Ensure truly free
2. **Check license** - Document license type
3. **Test quality** - Verify asset quality
4. **Add to library** - Document source

## License Notes

### CC0 (Public Domain)

- No attribution required
- Can use commercially
- Can modify
- Best for projects

### CC-BY (Attribution)

- Attribution required
- Can use commercially
- Can modify
- Check specific requirements

### MIT

- Attribution required (usually)
- Can use commercially
- Can modify
- Check specific requirements

### Always Check

- Individual item licenses may vary
- Some sources have mixed licenses
- Commercial use may have restrictions
- Attribution requirements vary
