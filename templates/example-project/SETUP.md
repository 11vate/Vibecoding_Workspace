# Example Project Setup

## Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Navigate to the example project
cd templates/example-project

# Install dependencies
npm install

# Start development server
npm run dev
```

The development server will start on `http://localhost:3000` and automatically open in your browser.

## Troubleshooting

### Port Already in Use
If port 3000 is already in use, Vite will automatically try the next available port.

### Service Worker Issues
Clear browser cache and reload if service worker doesn't register properly.

### TypeScript Errors
Run `npm install` again to ensure all type definitions are installed.
